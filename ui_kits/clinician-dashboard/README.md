# Clinician Dashboard — UI Kit

Desktop surface for eye-care professionals. Canvas 1280×820. 280px fixed sidebar, navy top bar, content in white cards on surface gray.

## Components
- `Sidebar` — 280px fixed nav with logo, sections, clinician chip
- `TopBar` — search + actions
- `PatientListRow` — sortable patient list item
- `ExamCard` — AI-assisted exam viewer tile
- `MetricTile` — dashboard stat
- `AIReviewPanel` — Doctor Eye AI findings
- `StatusPill` — severity indicator

## Screen (index.html)
Single-screen patient queue with an AI exam detail drawer. Demonstrates list → detail flow.
