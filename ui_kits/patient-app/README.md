# Patient App — UI Kit

Mobile-first patient surface. Canvas 390×844 (iPhone 13/14). Cards live on surface gray, content on white. Bottom tab bar is fixed.

## Components
- `TopBar` — scroll-aware app header with avatar + bell
- `HeroCard` — today's vision check prompt
- `StatCard` — clinical reading (IOP, acuity, etc.)
- `AppointmentRow` — upcoming appointment list item
- `TabBar` — 4-tab bottom navigation
- `PrimaryButton` / `GhostButton`
- `ResultBadge` — status pill

## Screens (index.html)
1. **Home** — greeting, today's check, quick stats
2. **Results** — latest exam summary with measurements
3. **Book** — appointment calendar
4. **Profile** — account settings

Use the tab bar at the bottom to navigate between screens.
